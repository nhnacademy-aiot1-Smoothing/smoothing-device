# 🛠️ 개발 내용
### 우혜승
- 디바이스에 관련된 생성, 조회, 수정, 삭제 요청이 들어오면 해당 작업을 처리합니다.
- Rule Engine과 연동되어 Broker와 관련된 요청이 들어오면 Rule Engine에서도 요청을 처리합니다.
